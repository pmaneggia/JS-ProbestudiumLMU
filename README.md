# JavaScript Übersicht 

Syntax ähnlich wie Java und C, objektorientiert, funktional, prototypische Vererbung, dynamisch typisiert.

## Kommentare
```javascript
// einzeilig
/* mehrzeilig: Zeile 1
Zeile 2 */
```

## Variablen
```javascript
x = 3; // global
let x = 3; // lokal
var x = 3; // lokal (altmodisch)
const X = 3; // Konstante
```

## Datentypen
```js
let s = "hallo"
```

## Debugging
```js
console.log("Hallo Welt");
```
